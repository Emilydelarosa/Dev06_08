# Dev06_08
Live coding application for Alterna Dev06 Back-end Fundamentals

## Setup

Download and install [.net 7](https://dotnet.microsoft.com/en-us/download/dotnet/7.0)

```bash
dotnet --version 
```

## Build
```bash
donet build
```

## Run Test
```bash
dotnet test
```
